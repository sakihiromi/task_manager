# タスク管理ダッシュボード - 実装計画

## 概要

可愛くて直感的なUIで、やる気が出るタスク管理ダッシュボードを作成します。以下の3つの主要カテゴリーを管理できます：

- **事務作業**: 日常的な事務タスク
- **研究タスク管理**: 実験、論文、情報収集など
- **プライベート**: 個人的な用事

達成度を可視化し、モチベーションを向上させる機能も実装します。

## 技術スタック

- **HTML**: 構造
- **CSS**: スタイリング（可愛いデザイン、アニメーション効果）
- **JavaScript**: インタラクティブ機能、データ管理
- **LocalStorage**: データの永続化

### 主要機能

#### 1. カテゴリー別タスク管理
各カテゴリーで以下の操作が可能：
- タスクの追加（タイトル、説明、期限、優先度）
- タスクの編集
- タスクの削除
- タスクの完了チェック

**研究タスク**には以下のサブカテゴリーを含む：
- 実験
- 論文
- 情報収集

#### 2. 達成度可視化
- **全体の達成率**: すべてのタスクの完了率を表示
- **カテゴリー別達成率**: 各カテゴリーの進捗を個別に表示
- **完了数カウンター**: 完了したタスク数を表示
- **進捗バー**: ビジュアルな進捗表示

#### 3. モチベーション向上機能
- タスク完了時のアニメーション効果（紙吹雪、キラキラなど）
- 達成率に応じた励ましメッセージ
- 可愛いアイコンとイラスト
- スムーズなトランジション効果

#### 4. データ永続化
- LocalStorageを使用してタスクデータを保存
- ページをリロードしてもデータが保持される

### デザイン方針

#### カラーパレット
- **プライマリカラー**: 明るいパステルピンク `#FFB6C1` - `#FF69B4`
- **セカンダリカラー**: ライトブルー `#A0D8F1` - `#6CB4E8`
- **アクセントカラー**: ラベンダー `#E0BBE4` - `#C9A0DC`
- **成功カラー**: ミントグリーン `#B4E7CE`
- **背景**: グラデーション（パステルピンク → ラベンダー → ブルー）

#### タイポグラフィ
- **英語**: Poppins（Google Fonts）
- **日本語**: Noto Sans JP（Google Fonts）
- フォントサイズは可読性を重視

#### アニメーション
- ホバー時の浮遊効果
- タスク完了時の紙吹雪エフェクト
- スムーズなフェードイン・フェードアウト
- バウンスアニメーション
